Text import widgets
===================

The common purpose of widgets of this category is to import text data in
Orange Canvas, either from the keyboard (:ref:`Text Field`), from files
(:ref:`Text Files`), or from the Internet (:ref:`URLs`). They all emit
*Segmentation* data.
