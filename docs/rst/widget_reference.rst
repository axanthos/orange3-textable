Widget reference
================

This part of the documentation explains the effect of every control of each
Orange Textable widget.

Orange Textable introduces mainly two new data types in Orange Canvas:
*Segmentation* and *Table*. Widgets making up Orange Textable are grouped
into 4 main categories based on their input and output data types:
