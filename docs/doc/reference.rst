Widgets
=======

This part of the documentation explains the effect of every control of
each Orange Textable widget. The Orange Textable’s widgets are grouped
into 4 main categories based on the type of functionality they offer.
The last section documents the details of JSON formats that can be used
for the configuration of some of Orange Textable’s widgets.
